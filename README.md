[200~# High-Throughput Financial Fraud Detection & Transaction Interception Platform

## Overview

This project simulates a production-grade financial transaction processing platform that ingests, validates, stores, transforms, and analyzes high-volume transaction data for fraud detection and AML (Anti-Money Laundering) use cases.

The project is being developed incrementally to mirror how real-world data engineering systems evolve.
